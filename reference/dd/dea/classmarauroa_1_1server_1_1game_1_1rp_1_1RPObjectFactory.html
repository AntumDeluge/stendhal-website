Detailed Description

This class is a factory to convert RPObjects into real objects for your game. This class is invoked by whatever that load an object into the server:

  • JDBCDatabase.

To make it work on your game you need to subclass and implement:

static RPObjectFactory getFactory()

The method will be called through refletions.
