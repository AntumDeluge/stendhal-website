Detailed Description

Creates a portable NPC who gives ALL players powerful items, increases their level and makes them admins. This is used on test-systems only. Therefore it is disabled in default install and you have to use this parameter: -Dstendhal.testserver=junk as a vm argument.

As admin uses /script AdminMaker.class to summon her right next to him/her. Please unload it with /script -unload AdminMaker.class
