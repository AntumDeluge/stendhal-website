Detailed Description

An RPLink represent an object relation that is not as strong as RPSlot, and although it could be modelled with a slot it would be better to manage it with an RPLink.

I.e: Buddies are handle now with a slot, when buddies are not an slot, so it just makes code harder and more prone to error. Buddies are not an slot in the sense that it is not something that a player contains, but an abstraction of something that a player is aware of.

Bag is an slot Buddies should be an RPLink
