◆ SPC

final String games.stendhal.client.gui.stats.StatsPanelController.SPC = "\r\u00a0"
staticprivate

A string used as a white space at the status labels. This is a combination of carriage return and no-break space, so that a possible automatic line break is inserted at the CR, and the next line starts with a leading space so that it is clear it belongs together with the previous line. In case the text fits in one line, it looks like a normal space.
