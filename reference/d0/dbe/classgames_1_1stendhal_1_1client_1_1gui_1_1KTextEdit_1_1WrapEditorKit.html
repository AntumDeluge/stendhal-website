Detailed Description

This is a workaround to line break behavior change between java versions 6 and 7. Long words do not get line breaks and no officially supported mechanism to get the old behavior is provided. Java bug 7125737 was closed as "Not an Issue".

The solution here is by StanislavL, published at multiple places, including here.
