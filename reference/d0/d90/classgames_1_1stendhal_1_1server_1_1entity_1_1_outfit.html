Detailed Description

A data structure that represents the outfit of an RPEntity. This RPEntity can either be an NPC which uses the outfit sprite system, or of a player.

You can use this data structure so that you don't have to deal with the way outfits are stored internally.

An outfit can contain up to nine parts: body, dress, head, mouth, eyes, mask, hair, hat, & detail.

Note, however, that you can create outfit objects that consist of less than nine parts by setting the other parts to null. For example, you can create a dress outfit that you can combine with the player's current so that the player gets the dress, but keeps his hair, head, and body. To denote that a layer should not be drawn, its value can be set to -1.

Not all outfits can be chosen by players.
