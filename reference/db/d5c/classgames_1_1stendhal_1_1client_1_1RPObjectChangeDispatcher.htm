Detailed Description

A dispatcher for RPObjectChangeListeners. This normalizes the tree deltas into individual object deltas.

NOTE: The order of dispatch between contained objects and when their container is very specific. Children objects are given a chance to perform creation/updates before their parent is notified it happened to that specific child. For cases of object removal, the parent is notified first, in case the child does destruction/cleanup.
