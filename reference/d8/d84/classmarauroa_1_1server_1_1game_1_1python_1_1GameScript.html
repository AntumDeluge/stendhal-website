Detailed Description

This class is a wrapper for calling python in a better way. All the jython specific code is here.

You are expected to define the file that contains your python script at server.ini in the field python_script

For example:

  python_script=foo.py
  python_script_world=MyPythonRPWorld
  python_script_ruleprocessor=MyPythonRuleProcessor
