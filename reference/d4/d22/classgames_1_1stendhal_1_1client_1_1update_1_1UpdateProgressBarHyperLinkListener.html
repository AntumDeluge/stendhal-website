Detailed Description

Opens Hyperlinks in an external browser using the Bare Bones Browser Launcher.

Note: This is a copy of BareBonesBrowserLauncher because classes in the update package must not depend on anything outside this package.

Bare Bones Browser Launch for Java
Utility class to open a web page from a Swing application in the user's default browser.
Supports: Mac OS X, GNU/Linux, Unix, Windows XP/Vista/7
Example Usage:
    String url = "https://www.google.com/";
    BareBonesBrowserLaunch.openURL(url);
