Detailed Description

This class is the glue to Marauroa, it extends ClientFramework and allows us to easily connect to an marauroa server and operate it easily.

items contains a list of names and timestamp. That information can be used to decide if a transfer from server is needed. By setting attribute ack to true in a TransferContent it will be acknowledged. All acknowledges items in the returned List, will be transfered by server.

Parameters
