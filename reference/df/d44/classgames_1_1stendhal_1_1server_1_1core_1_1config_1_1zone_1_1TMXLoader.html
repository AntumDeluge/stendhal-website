Detailed Description

Loads a TMX file to server so it can understand: a) The objects layer b) The collision layer c) The protection layer. d) All the layers that are sent to client e) The tileset data that is also transfered to client f) A preview of the zone for the minimap.

Client would get the layers plus the tileset info.
