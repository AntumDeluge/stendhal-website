Detailed Description

The ServerInfo message is sent from server to client to inform client about any relevant info the server has to transmit. They are in the form of <attribute>=

This message also sent to client the list of RPClass that are contained at server at the time the message was sent. It is because of that, that RPClass definition must be load previously to game start.
