Detailed Description

Class for determining the used transparency mode. The behavior depends on the setting ui.transparency. Mode "bitmask" means bitmask transparency, and "translucent" means that full alpha transparency is used. Mode "auto", means that the appropriate mode is decided based on a speed test.
