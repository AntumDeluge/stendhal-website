Detailed Description

Blend composer for removing effect of color multiply. This is yet another Stendhal specific blend mode that does not have an established name. Both the images are supposed to have been multiplied by the same color.

Blend 2 pixels. Removes the effect of color multiplication on the underlaying image, using the brightness of the original version of the overlaying image as the amount of bleaching. The brightness of the overlaying image is adjusted so that the color of each pixel is assumed to be result of (yet another) multiplication with color A. The added brightness to the underlaying image is multiplied with the hypothesized color A, so that the overall effect looks like colored light.

srcPixelupper pixel color data
dstPixellower pixel color data
